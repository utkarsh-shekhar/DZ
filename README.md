# DZ 
Pronounced as dizzy, DZ A virtual pet app powered by libGDX, developed using Android Studio.
The authors are new to git and everything and may not entertain all the contributions.

Running
* [Setup your development environment](https://github.com/libgdx/libgdx/wiki)
* Clone the repository or download and extract the ZIP file
* Import the project into your preferred development environment, run, debug and package it!
  * [Eclipse](https://github.com/libgdx/libgdx/wiki/Gradle-and-Eclipse)
  * [Intellij IDEA](https://github.com/libgdx/libgdx/wiki/Gradle-and-Intellij-IDEA)
  * [NetBeans](https://github.com/libgdx/libgdx/wiki/Gradle-and-NetBeans)
  * [Commandline|Gradle on the Commandline](https://github.com/libgdx/libgdx/wiki/Gradle-on-the-Commandline)
